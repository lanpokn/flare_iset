Design custom CFA patterns and colors


The cfaDesignUI utility can be used to specify custom Color filter arrays (CFA). A CFA is specified in terms of its spatial arrangement and the transmittances of the color filters. 

cfaDesignUI allows CFA designs that are formed by repeating a core CFA pattern. The number of rows columns and colors in this basic building block are specified by the user.  In the UI window each color has its own dedicated panel. The sample points for each color can be specified by clicking on the nRows x nColumns array of push buttons. When a particular location is selected for a particular color, the pushbutton is colored white. The same location can not be selected for more than one color. Colors are specified in two ways:

1.  As Gaussian curves along the wavelength range (380 -- 1068 nm)

2. Imported from an existing ISET coded color filter file
